# SQL Data Analysis Internship - Task 4

Analytical reports using JOINs, GROUP BY, HAVING and aggregate functions.

## Project
In this task, I created different SQL reports using the Student Management Database. I used JOIN, GROUP BY, HAVING, aggregate functions, and simple SQL queries to analyze student performance and course data.

## query 1 - Average Grade by Gender : This report shows the average grade of male and female students.

- Summary:
- It helps compare the overall performance of male and female students.
- The average grade is calculated using the data available in the database.

## query 2 - Pass Rate Per Course : This report shows the percentage of students who passed each course.

- Summary:
- In Mathematics, **85.71%** of the students passed.
- In Science, **92.86%** of the students passed.
- In English, **100%** of the students passed.
- Mathematics had the lowest pass percentage, while all students passed English.

## query 3 - Top 3 Students : This report finds the top three students based on their overall average marks.

- Summary:
- The report lists the students who performed the best across all courses.
- These students have the highest average grades in the database.

## query 4 - Students Enrolled in Multiple Courses : This report displays students who are enrolled in more than one course.

- Summary:
- Every student is enrolled in all three courses.
- Each student is taking Mathematics, Science, and English.

## query 5 - Improvement Report : This report compares Mathematics and English scores.

- Summary:
- Some students scored better in English than in Mathematics.
- This report can be used to identify improvement in performance.

## SQL Concepts Used

- INNER JOIN

- Aggregate Functions (AVG, COUNT, MAX)
